# 🤟 Sign Language Detection System

A real-time Sign Language Detection application built using Python, Streamlit, OpenCV, and MediaPipe. The system captures hand gestures through a webcam, detects hand landmarks, and recognizes predefined signs instantly.

## 🚀 Features

- Real-time hand gesture detection
- Hand landmark tracking using MediaPipe
- Live webcam integration
- Streamlit-based interactive UI
- Visual hand skeleton visualization
- Fast and lightweight gesture recognition

## ✋ Supported Gestures

- ✊ FIST
- ✋ FIVE
- ☝ ONE
- ✌ VICTORY
- 👍 THUMBS UP
- 🤟 LOVE YOU
- L

## 🛠️ Tech Stack

- Python
- Streamlit
- OpenCV
- MediaPipe
- Streamlit WebRTC

## 🔄 Workflow

```text
Webcam Input
      ↓
Capture Video Frame
      ↓
Hand Detection using MediaPipe
      ↓
Extract 21 Hand Landmarks
      ↓
Analyze Finger Positions
      ↓
Gesture Recognition
      ↓
Display Hand Skeleton & Gesture Name
      ↓
Real-Time Output
```
